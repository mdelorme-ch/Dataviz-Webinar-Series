#30DayMapChallenge 2024

2024

This is a collection of maps I created during the #30DayMapChallenge in November 2024.

The challenge encourages map enthusiasts to create and share a new map every day, following a specific prompt for each day. It is an exciting opportunity to learn new techniques, experiment with data visualization, and join a vibrant community of cartographers and data enthusiasts.

The challenge was founded and has been organized since by Topi Tjukanov. Visit his website for the challenge: 30daymapchallenge.com

Click on each map to find out more details. The tools used are listed for each map. You can find the code for the maps I created with R in this Github repo.
